1. В поле projectName пишем название проекта
2. В поле behaviourTemplate пишем минимальный поведенческий шаблон для чата (какую роль он будет отыгрывать в беседе)
поле extraInfoForChatPath не трогаем! папка будет синхронизироваться с гугл диском
3. В поле websitesLinks передаем объектики вот такого вида через запятую:
  	{
                link: 'https://5controls.com/',
                crawlDepth: 5,
            },
  проще всего будет копировать уже существующий и менять в нем ссылку и глубину краулинга ( но эта фича пока плохо работает, поэтому временно будет отключена
 4. в поле youtubeVideoLinks через запятую вставляем ссылки на видео в ютубе в одинарных кавычках
5. В поле gitHubRepositories  логика такая же, как и в поле  websitesLinksЖ  
 	{
                link: 'https://github.com/easably/games-experiments',
                branch: 'master'
            }
            передаем такие объекты через запятую, указываем там ссылку на репозиторий на гитхабе и название ветки
